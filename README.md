# Insta shop bot

## Requirements
* Mozilla
* python 3.7
* pipenv

## How to use
1. Rename `.env.example` to `.env`
2. Change credentials
3. Create virtual environment
4. Activate virtual environment
5. Install dependencies
6. [Windows Task Scheduler](https://instapy.org/automate-instapy)
